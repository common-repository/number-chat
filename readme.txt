== Description ==
[Demo](https://wpcohort.io/number-chat/)<br />

NumberChat for WordPress allows you to add a sticky or a floating button that will take your site visitors from your website to your WhatsApp with just a couple of clicks.

NumberChat allows your visitors to type the text message before sending visitors to WhatsApp. 

It's an essential tool nowadays to connect your website to WhatsApp as it helps generate leads which is the key to every business success.

Using NumberChat, you can configure Name, Avatar, and Representative job title.

== Installation ==
Manual installation is easy and takes fewer than one minute.

Download the plugin from wordpress.org, unpack it and upload the [ Number Chat] folder to your wp-content/plugins/ directory.
Activate the plugin through the ‘Plugins‘ menu in WordPress.
Go to your main WordPress menu > Number Chat to add a new WhatsApp account.
Go to Floating Widget to select the WhatsApp profile to display on your website.
